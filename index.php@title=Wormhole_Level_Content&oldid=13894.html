NOTE: Wormhole effects affect player ships, so one should take that into consideration before attempting to create fits for this content. Wormhole effects do not, however, affect the NPCs in the sites.

For example, it is possible to run C6 Wolf-Rayet sites with T3-destroyers with logistics support.
