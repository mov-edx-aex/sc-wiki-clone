How To Share a Fit on Discord[edit]

There are two ways to post a fit on Discord. Fits are usually shared in our #fitting-lab channel.

Note 1: Method One allows you to share any fit (Personal, Corporation, or Community fittings) while Method Two allows you to only share Personal fittings.
Note 2: Unlike Method One, Method Two fits cannot be easily copied from Discord and then imported into EVE or other programs by others unless you use the eft option (see below).

Method One[edit]

Step 1: Copy Your In-Game Fit[edit]

Step 2: Paste and Wrap It[edit]

Paste your fit into a Discord channel and then wrap it with three backticks (```).

Example:[edit]

```
[Buzzard, SC Buzzard]

Inertial Stabilizers II
Power Diagnostic System II
Warp Core Stabilizer II

Cargo Scanner II
Relic Analyzer II
Data Analyzer II
Scan Rangefinding Array II
5MN Y-T8 Compact Microwarpdrive

Core Probe Launcher II
Covert Ops Cloaking Device II
Interdiction Nullifier I

Small Ancillary Current Router II
Small Gravity Capacitor Upgrade I

Sisters Core Scanner Probe x16
```

This will produce: 

[Buzzard, SC Buzzard]

Inertial Stabilizers II
Power Diagnostic System II
Warp Core Stabilizer II

Cargo Scanner II
Relic Analyzer II
Data Analyzer II
Scan Rangefinding Array II
5MN Y-T8 Compact Microwarpdrive

Core Probe Launcher II
Covert Ops Cloaking Device II
Interdiction Nullifier I

Small Ancillary Current Router II
Small Gravity Capacitor Upgrade I

Sisters Core Scanner Probe x16

Method Two[edit]

Use Allison bot's /fit slash command in Discord. Note that this slash command ONLY works for your Personal fittings and NOT Corporation or Community fittings.

To use:

  • First, type /fit and hit the Spacebar.
  • Next, type the Fitting Name of the ship you'd like to share (it doesn't have to be exact as fuzzy search logic is used) and then press Enter

  • Enjoy with others!

Note: Fits shared with the default slash command cannot be easily copied from Discord and then imported into EVE or another application unless you use the eft option (see below).

Options[edit]

The /fit slash command has two options: id and eft. Note that the id option and the eft option can be used separately or at the same time!

Using the id Option[edit]

The id option is useful if you can't remember the full name of your in-game fit or your search returns multiple entries.

To use:

  • Type /fit and hit the Spacebar.
  • Type the partial Fitting name of the ship you'd like to share (for instance "buzz") and then press Enter

  • Allison will return a list of your matching fits along with a numerical id

  • Now search again using /fit and the partial name of your fit, but now include the optional id number of the ship you'd like to share (in this example 1) and then press Enter.
    • Note, to bring up this optional menu you must first click the +2 more text.

  • Enjoy!

Using the eft Option[edit]

The eft option allows your fit to be displayed in eft format for easier sharing (just like Method One).

To use:

  • Type /fit and hit the Spacebar.
  • Type the Fitting name of the ship you'd like to share, but now include the optional eft set to True and then press Enter
    • Note, to bring up this optional menu you must first click the +2 more text.

  • Others can now copy and easily import and share your fit with others. Enjoy! üçª
