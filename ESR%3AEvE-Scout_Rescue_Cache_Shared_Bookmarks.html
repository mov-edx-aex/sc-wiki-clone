Shared Folders[edit]

There are six folders for accessing cache bookmarks and one folder for new bookmarks.

All Signal Cartel members have View access to the cache bookmark folders and Use access to the ESRC New Bookmarks folder. This ensures no one is able to copy our entire collection of cache bookmarks for nefarious purposes.

Members of the ESR Coordinator team and select members of the 911 Operator team are able to move bookmarks from ESRC New Bookmarks to the correct folder.

The seven Bookmark folders are structured as follows:

  • ESRC New Bookmarks (all new bookmarks for newly sown caches, or updated locations, go here)
  • ESRC Jxxxx0x
  • ESRC Jxxxx1x
  • ESRC Jxxxx2x
  • ESRC Jxxxx3x
  • ESRC Jxxxx4x
  • ESRC J---other

The folders are divided up as such due to the limitations imposed by CCP on the maximum number of bookmarks any shared folder can contain. This subdivision allows us to cover all the 2604 needed Wormholes in an easy to way categorization system.

How to Find and Connect to the Shared Bookmark Folders[edit]

  1. Links to the seven folders are located in our in-game Corp Bulletins by navigating here:
    Neocom Menu → Social → Corporation → My Corporation (tab) → Bulletins (subtab) under "B5: EvE-Scout Rescue Shared Bookmarks".
  2. Connect to each folder:
    1. Left click on each folder name and then select "Connect" (Note that it doesn't matter if you have the "Online Location Folder" radio button or "Offline Location Folder" radio button selected - the folders will be saved either way).

How to Use the Shared Bookmark Folders[edit]

  • Upon entering a J-space (wormhole) system, Signaleers must online the appropriate ESRC Shared Bookmark folder that contains the cache bookmark for that particular system.
    • For example: If you are in J145225, you would online the "ESRC Jxxxx2x" folder.
    • To online a folder: Right click on the appropriate folder and select "Online Folder".
    • You will now be able to use the cache bookmark to warp directly to the cache's location.

Protip: Press the "L" key to activate the "Local Locations" shortcut which will pop up a window of all your bookmarks (including that of the cache) in the current system!

  • Note that a maximum of three Shared folders can be online at any given time. Therefore, Signaleers will need to manage their Shared folders by bringing folders online and offline as appropriate.
    • To offline a folder: Right click on the appropriate folder and select "Offline Folder".

Guidelines[edit]

  • The use of these bookmarks to find our rescue caches is completely optional. However, recording/saving of a new bookmark is not optional.
  • Please add a bookmark to the ESRC New Bookmarks folder if the cache has moved to a new location. The ESR Team will remove the duplicate with the old location.
  • Bookmarks should be named using the copiable bookmark text found in Allison.
  • Do not add a bookmark to the ESRC New Bookmarks folder if there is already a bookmark for the same location.
  • Because of the level of verification required to ensure the ESRC New Bookmarks folder is both current and accurate, please only add cache bookmarks to this folder.
    • Do not bulk add from your existing store.
