Note: Signal Cartel's Little Helper is no longer being maintained and can sometimes be slow.  If this happens, give localthreat a try.

Signal Cartel's Little Helper is a web-based tool that allows users to quickly determine the danger level of a system by giving an overview of each pilot's PVP history.

Signal Cartel's Little Helper was designed by fellow Signaleer Portia Tigana and inspired by the now depreciated Pirate's Little Helper.

To Use:

  • Select all the pilots from your local in-game chat window
  • Copy
  • Paste them into the tool
  • Click "Submit"

You will then see an overview of each pilot's PvP history and other useful information. The data displayed is pulled from CCP and zKillboard.com.
