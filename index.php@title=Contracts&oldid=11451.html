Contracts Overview[edit]

Contracts in EVE are mostly "Item Exchange Contracts" or "Courier Contracts". In turn, these can either be assigned to a pilot, corporation or the public.

There are two main ways to create a contract in EVE:

  1. Neocom Menu → Finance → Contracts and click Create Contract
  2. Selecting the items in your item hangar and then right clicking and selecting Create Contract

Afterwards, you will be presented with a 4-step process with numbered windows (1/4, 2/4, 3/4, 4/4).

Creating a Generic Contract[edit]

Select Contract Type (1/4) Window[edit]

In the first window (1/4) you will be asked if you would like to make an Auction, Courier, or Item Exchange contract. You will also be asked whether you wish to make it "Public" or "Private". If you make it private you must type in the name of a player, corporation or alliance.

Pick Items (2/4) Window[edit]

This is an item selection window. If you highlighted some items and clicked "Create Contract", then these will already be selected. You can review and add/remove items by selecting or deselecting them, respectively.

Select Options (3/4) Window[edit]

For Courier Contracts:[edit]

  • "Ship To" This is the station you would like the courier to take your goods to.
  • "Reward" is how much the courier will get paid. This is taken from your ISK wallet when you create the contract.
  • "Collateral" is how much ISK the courier will have to place as a deposit when hauling your goods. If the courier fails to deliver the goods in the agreed time, or if the courier package is broken open, then you will get the collateral back instead of the goods.
  • "Expiration" date is how long the contract will be available to accept.
  • "Days to complete" once the contract has been accepted, how long they will have to complete it.
  • The "Description (optional)" box allows you to add any extra information which might be useful.

For Item Exchange Contracts:[edit]

  • This window will ask you to select how much money you will pay or receive as part of the contract.
  • Generally speaking, you will ignore the "I will pay" box and enter a value in the "I will receive" box.
  • Set the "Expiration". After this time, the contract will expire - but to get the items back you will need to open the contract and delete it.
  • The "Description (optional)" box allows you to add any extra information which might be useful.

Confirm (4/4) Window[edit]

This is the confirmation window. Read it carefully before you click finish.

Specific Instructions for SCRUBS[edit]

  • When submitting blueprints to Cevax Aubaris for appraisal:
    • Please use an Item Exchange Contract.
    • The contract should be "Private" and assigned to Cevax Aubaris.
    • "I will pay" should be set to 0.00
    • "I will receive" should be set to 0.00
    • "Expiration" should be set to 4 weeks.
    • If you wish, you may write "donation" or "auto-accept" in the "Description (optional)" box and the contract will be treated as such.
  • After submitting the blueprints, you will receive a reply with a proposed price.
  • If you accept the offer, the ISK will be transferred to you.

Please see the SCRUBS page for more info.

Specific Instructions for SCRAMS[edit]

  • Before making your move request, repackage any ships that don't have rigs.
  • To submit a SCRAMS request:
    • Please use a Courier Contract.
    • The contract should be "Private" and assigned to Crikey Freight.
    • "Ship To" should be one of the Signal Cartel Stations.
    • "Reward" should be 0.00 unless an amount of ISK has been agreed upon with Xavec.
      • If the reward it is not zero, this is considered a tip for the courier.
    • "Collateral" should be maximum 110% the value of the goods being transported.
      • The collateral is the full extent of the haulers liability. If you choose to set the collateral at zero, then you are accepting no reimbursement if your cargo is lost.
    • "Expiration" should be set to 2 weeks
    • "Days to complete" should be set to 14 days
    • In the "Description (optional)" box - Please write "SCRAMS", so our pilots can properly route the contract.

Please see the SCRAMS page for more info.
